# Crypto Forecast Service

Web-сервис для прогнозирования стоимости криптовалют с использованием современных моделей машинного обучения и глубокого обучения.

Проект объединяет исследование моделей прогнозирования временных рядов, REST API для получения прогнозов и веб-интерфейс для их визуализации.

## Возможности

- прогнозирование стоимости криптовалют на **30** и **180** дней вперед;
- сравнение различных моделей прогнозирования;
- использование ансамблей моделей;
- прогнозирование волатильности с помощью GARCH;
- автоматическое обновление рыночных данных;
- просмотр исторических прогнозов;
- отображение метрик качества моделей;
- REST API для получения прогнозов;
- web-интерфейс на Streamlit.

---

## Используемые модели

Для прогнозирования исследованы следующие модели:

- LSTM
- GRU
- TCN
- Chronos
- TimesFM
- XGBoost
- GARCH

В качестве основных моделей используются гибридные ансамбли:

- **XGBoost + LSTM**
- **XGBoost + TCN**

Они показали наилучшие результаты по метрикам **MAE** и **MAPE**.

## Как работает сервис

1. Получение актуальных рыночных данных с криптобиржи.
2. Сохранение данных в SQLite.
3. Загрузка обученных моделей.
4. Формирование прогнозов на 30 и 180 дней.
5. Расчет доверительных интервалов с помощью GARCH.
6. Сохранение прогнозов в базе данных.
7. Отображение результатов через REST API и Streamlit.

---

## REST API

| Метод | Endpoint | Назначение |
|--------|----------|------------|
| GET | `/api/cryptocurrencies` | список криптовалют |
| GET | `/api/forecasts/{symbol}` | текущие прогнозы |
| GET | `/api/historical/{symbol}` | исторические данные |
| GET | `/api/forecast-history/{symbol}` | история прогнозов |
| GET | `/api/metrics/{symbol}` | метрики моделей |
| POST | `/api/admin/update` | обновление данных и прогнозов |

---

## Метрики

Для оценки качества моделей используются:

- MAE
- MAPE

Сервис хранит историю качества моделей и позволяет сравнить прошлые прогнозы с фактическими значениями.
